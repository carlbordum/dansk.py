# dansk.py

Python, but danish.


## Installering

For at installere `dansk.py`, skal du køre: :

```sh
    $ pip install dansk
    $ python -m dansk installér
```


## Anvendelse

Alt bliver godt, hvis du skriver `# coding=dansk` på den første linje i
din kode.

Se eksempler i `eksampler/` mappen.
